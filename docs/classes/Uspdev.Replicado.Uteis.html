semestre()

semestre(string  $data_string = null) : Array

Determina a data de início e a data de fim do semestre quer contém $data_string.

Se $date não for informado, utilizará a data corrente do sistema. $data_string pode ser em qualquer formato aceito por DateTime para criar uma data

Parameters

string $data_string

(opcional) Data na qual vai buscar os limites do semestre.

Returns

Array —

formato ['yyymmdd', 'yyyymmdd']

Examples

= Uteis::semestre();
** File not found : $inifim **
= Uteis::semestre('2019-10-20');
** File not found : $inifim **
